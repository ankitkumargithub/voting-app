User Authentication:

/signup: POST - Create a new user account
/signin: POST - Login to an existing account. [aadhar card no + Password]

Voting:
/canndidate: GET - Get the list of canndidate
/vote/:candidate ID: POST - Vote for an specific candidate

/Vote Counts: 
/Vote/counts: GET - Get teh list of candidate sorted by thier vote counts.

User/Profile:

/profile: - GET - Get the user's profile information
/profile/password: PUT - Change the user's password.

Admin Candidate Management:

/candidates/:POST - Create a new candidate.
/candidate/:candidateId/: PUT - Update and existing candidate
/candidate/:candidateId/:Delete - Delete a candidate from the list.
